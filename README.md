# Summary Buddy Chatbot

## Setup

### Install Dependencies
- node
- npm
- [nx](https://nx.dev/docs/getting-started/installation)
- python3
- [uv](https://docs.astral.sh/uv/)
- [docker](https://www.docker.com/)
- [Docker Compose v2.20+](https://docs.docker.com/compose/)

### Run project

```bash
nx run-many -t up
```
