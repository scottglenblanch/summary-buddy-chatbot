# Summary Buddy Chatbot

## Setup

### Install Dependencies
    - node
    - npm
    - python3
    - [uv](https://docs.astral.sh/uv/)
    - [docker](https://www.docker.com/)
    - [Docker Compose v2.20+](https://docs.docker.com/compose/)

### Install Npm and Python Packages

```bash
nx run-many -t install -p webui,api-server --parallel=2
```
